# dotfiles
> :wrench: .files: zsh, oh-my-zsh, z, git, node.js, python, atom, ...

## Prerequisites

- [Git >= 2](https://git-scm.com/)
- [Node.js >= 5](http://www.zsh.org/)

## Install

Clone this repository in **~/.dotfiles**
```bash
git clone https://github.com/mlcdf/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
```

Install programs
```console
$ sudo ./bootstrap
```

Symlink files
```console
$ ./install
```
